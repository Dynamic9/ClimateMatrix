# Dynamic 9

Engineering, AI & Robotics consulting website.

## Structure

```
index.html          — Landing page (services, process, CTA)
pages/
  tools.html        — Engineering tools & calculators hub
  about.html        — Company mission and values
  contact.html      — Contact form
css/
  style.css         — Full stylesheet
js/
  main.js           — Navigation, animations, form handling
assets/             — Images and media (add as needed)
```

## Deployment (Hostinger)

This is a static HTML/CSS/JS site. To deploy on Hostinger:

1. Log in to your Hostinger control panel
2. Go to **File Manager** (or use FTP)
3. Navigate to `public_html`
4. Upload all files and folders from this repo, preserving the directory structure
5. `index.html` will be served as the homepage automatically

## Development

No build step required. Edit the HTML/CSS/JS files directly and upload to your hosting.
